# RAG Book Project

This project implements a Retrieval-Augmented Generation (RAG) pipeline
for answering questions based on a historical text.

## Files
- `proyecto.py`: Main execution script
- `vector.py`: Vector store creation and retrieval logic
- `evaluation.py`: Evaluation logic
- `evaluated_text.txt`: Text used for evaluation

## How to run
```bash
python proyecto.py
```
